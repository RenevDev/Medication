# Renev Medication

**Be reminded quickly and easily to take your medication, in a beautiful, well-organized app.**

Renev Medication is a modern, intuitive Android app designed to help you remember all your daily medication intake times, so you never forget a medicine. The app is built entirely according to the latest Material 3 Expressive design guidelines, ensuring a calm, focused, and highly accessible user experience.

## Key Features

* **The Medicine Cabinet:** Manage all your medications easily in one central place. Enter the names of your medications, specify the dosages, and set the desired intake times.
* **Today's Overview:** The home screen shows you at a glance which medications you need to take today and at which specific times. Taking it a little later today? No problem, adjust it.
* **Privacy comes first:** Your medical data belongs to you. All data and log files are stored 100% locally on your own device. No medical data is sent to or shared with external servers. An important addition is that if you connect to the Renev Medication Servers, you must always first verify that you want to do so.

## Installation Guide

Because the application is currently distributed via GitHub and is in the testing phase, you can install the application manually using the APK file. Follow these steps:

1. **Download the APK:** Go to the releases section on the right side of this page and download the latest `.apk` file.
2. **Open the file:** Tap the downloaded file on your Android device.
3. **Allow unknown sources:** For security reasons, Android asks if you want to allow installations from your browser or GitHub. Tap *Settings* in the pop-up window and enable the *"Allow from this source"* option.
4. **Complete the installation:** Return to the installation screen and tap **Install**. You can now use the app.
